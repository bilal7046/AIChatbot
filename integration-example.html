JMM Chatbot Widget - Integration Example

This page demonstrates how to integrate the chatbot widget into an existing website.

Basic Integration

Add this code before the closing </body> tag:

<!-- Load the chatbot widget -->
<script src="widget/chatbot-widget.js"></script>
<script>
    // Initialize with default settings
    window.JMMChatbot = new ChatbotWidget();
</script>

Custom Configuration

Customize position, color, and knowledge base:

<script src="widget/chatbot-widget.js"></script>
<script>
    window.JMMChatbot = new ChatbotWidget({
        position: 'bottom-left',           // Options: 'bottom-right', 'bottom-left', 'top-right', 'top-left'
        primaryColor: '#28a745',            // Your brand color (hex)
        knowledgeBase: 'widget/knowledge-base.json'  // Path to knowledge base
    });
</script>

Without Knowledge Base File

If you don't provide a knowledge base path, the widget uses built-in default responses:

<script src="widget/chatbot-widget.js"></script>
<script>
    // Uses default knowledge base
    window.JMMChatbot = new ChatbotWidget({
        position: 'bottom-right',
        primaryColor: '#007bff'
    });
</script>

Notes

  • The widget is self-contained and doesn't require any CSS frameworks
  • All styles are injected automatically
  • The widget works on all modern browsers
  • Voice input works best in Chrome and Safari
  • No dependencies or build process required
